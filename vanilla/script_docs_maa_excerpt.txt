=== ЭФФЕКТЫ ===

create_maa_regiment — Create MaA regiment for a ruler. If title is set, MaA will be created as title troops
  scope:ruler = {
    create_maa_regiment = light_footmen
    create_maa_regiment = {
      type = light_footmen        # mutually exclusive with type_of
      type_of = scope:maa_regiment
      check_can_recruit = no
      title = scope:title
      size = 5
    }
  }

change_maa_regiment_size — Change size of MaA regiment BY the specified value.
  Extra troops are deleted, by default new troops will have to reinforce, unless reinforce = yes
  scope:regiment = { change_maa_regiment_size = scope:change_by }
  scope:regiment = { change_maa_regiment_size = { size = 5 reinforce = yes } }

change_maa_troops_count — Change current troops count (can't exceed max)
destroy_maa_regiment — Destroy MaA regiment (не работает на полки наёмников/орденов)
  scope:regiment = { destroy_maa_regiment = yes }

Итераторы (character): every_/ordered_/random_/any_maa_regiment
  ordered_maa_regiment = { limit = {...} order_by = script_value position = int
    min = int max = script_value check_range_bounds = no <effects> }
Титульные: *_controlled_title_maa_regiment, *_owned_title_maa_regiment, *_title_maa_regiment
Армия: *_army_maa_regiment (include_hired = yes по умолчанию)
transfer_owned_maa_control / transfer_title_maa_ownership / transfer_title_maa
